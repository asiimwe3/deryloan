# DeryLoan 💰

> Microfinance Management System — built on OpenCBS Desktop architecture, modernised as a web app by Derycode.

## Features

Built directly from the OpenCBS source (https://github.com/OpenCBS/OpenCBS-Desktop):

1. Dashboard — portfolio stats, PAR alerts, monthly collections chart
2. Client Management — individuals, groups, companies with district filter
3. Loan Portfolio — full lifecycle: application → disbursement → repayment → closure
4. Repayments — record payments, due-today list, 12-month installment schedule
5. Loan Products — Business (18% flat), Agriculture (15% declining), Emergency (3%/month)
6. Savings — voluntary, compulsory and fixed deposit accounts
7. Accounting — journal bookings for disbursements, repayments and interest
8. Collaterals — land titles, vehicles and guarantors
9. Reports — Portfolio At Risk (PAR) breakdown by category
10. Settings — organisation configuration

## Tech Stack

- Pure HTML5 / CSS3 / Vanilla JavaScript (no framework dependencies)
- Designed for Uganda MFIs — UGX currency, Ugandan districts
- Supabase-ready backend integration (coming soon)

## Live Preview

Open index.html in any browser — no build step required.
